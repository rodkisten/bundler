# Third-party notices

This implementation was produced as a clean TypeScript port informed by the public behavior and interfaces of RodEruda/Eruda and by the source supplied with this task.

No source package from Luna, Licia, Eustia, or Chobitsu is copied into this directory and none is imported at runtime. Equivalent behavior is implemented with browser APIs and original local modules.

The only imports outside `src/devtools/` point to the repository's local Cipo and Fábrica packages.
